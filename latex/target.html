Descriptive statistics in Python
Many choices; a few methods in order of simplicity:
With scipy: scipy.stats.describe(...)
With Python 3 statistics: statistics.mean(...),
statistics.stdev(...), ...
With numpy (a bit old): data.mean(), data.std(), ...
These work ﬁne for “small data”, but we need to do better for
Big Data.
Simple statistics with PySpark
We need to put the data into an RDD, accessed through
Spark.
>>> from linalg import Vectors
>>> rdd = sc.parallelize([Vectors.dense([2, 0, 0, -2]),
... Vectors.dense([4, 5, 0, 3]),
... Vectors.dense([6, 7, 0, 8])])
>>> cStats = Statistics.colStats(rdd)
>>> cStats.mean()
array([ 4., 4., 0., 3.])
>>> cStats.variance()
array([ 4., 13., 0., 25.])
